# Ibex Reference Model

## Quick Start

- Clone the repository:
    ```
    git clone
    ```

- From the root directory, build with:
    ```
    make clean build
    ```

- From the root directory, run with:
    ```
    make run
    ```

- The initialized value of Register File, IMEM and DMEM can be changed in:
    - `register_file.mem`
    - `imem.mem`
    - `dmem.mem`
